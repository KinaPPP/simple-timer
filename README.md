# シンプルタイマー

Chrome拡張機能のアイコン部分にカウントダウンを直接表示するシンプルなタイマーです。

![バージョン](https://img.shields.io/badge/version-1.2-green)

## 特徴

- **アイコン表示** — ツールバーのアイコンに残り時間を常時表示（分 → 残り1分未満は秒）
- **シンプル操作** — テンキー入力でサクッとセット、スタートでポップアップが閉じる
- **一時停止 / 再開** — ポップアップを開いてディスプレイをクリックするだけ
- **サウンド通知** — 残り10秒に1回、終了時に3回鳴らして知らせる
- **ダークモード対応** — ポップアップのトグルでアイコン文字を白くできる
- **自動リセット** — 終了後1分で自動的に初期状態へ
- **ブラウザ再起動後も復元** — カウント中にブラウザを閉じても再開

## アイコン表示の仕様

| 状態 | 表示 | 色 |
|------|------|----|
| 待機中 | ST | ティール（ダークモード時：白） |
| カウント中（1分以上） | 残り分数 | ティール（ダークモード時：白） |
| カウント中（1分未満） | 残り秒数 | 赤 |
| 一時停止中 | 分数 or 秒数 | グレー |
| 終了 | 00 | 赤 |

## インストール方法

Chrome ウェブストア未公開のため、デベロッパーモードで読み込んでください。

1. このリポジトリをクローンまたはZIPでダウンロード
2. Chromeのアドレスバーに `chrome://extensions/` を入力
3. 右上の「デベロッパーモード」をオン
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## 動作環境

- Google Chrome（Manifest V3対応）
